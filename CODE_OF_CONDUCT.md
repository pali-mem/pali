# Code of Conduct

## Our Standard

Be direct, respectful, and technically honest.

Expected behavior:

- discuss code and behavior, not people
- assume good intent but be precise about problems
- keep review comments actionable
- avoid harassment, abuse, and exclusionary language

## Enforcement

Project maintainers may edit, remove, or reject contributions and comments that violate this standard.

For serious issues, report privately through the security or maintainer contact path instead of escalating in public threads.
